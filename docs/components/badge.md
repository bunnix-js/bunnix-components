# Badge

```js
import { Badge } from "@bunnix/components";

Badge({ tone: "success", size: "small" }, "Active");
```

Props:

- `tone`: base | success | info | warning | danger | accent | dimmed
- `variant`: solid | soft | outline
- `size`: xsmall | small | regular
- `shape`: capsule | circle
