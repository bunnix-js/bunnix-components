# Icon

```js
import { Icon } from "@bunnix/components";

Icon({ name: "star", fill: "base", size: "regular" });
```

Props:

- `name`: icon slug (e.g. `"star"`)
- `fill`: `base | white | secondary | tertiary | quaternary | destructive`
- `size`: `xsmall | small | regular | large | xlarge`
