# Utilities

Common utilities you can use to build custom components:

- Layout: `row-container`, `column-container`, `grid-flow`, `gap-xs|sm|md|lg`, `items-*`, `justify-*`
- Surfaces: `box`, `box-sm`, `box-control`, `box-capsule`, `card`, `shadow`, `rounded*`
- Buttons: `btn`, `btn-flat`, `btn-outline`, `btn-destructive`, `btn-lg`, `btn-xl`
- Forms: `input-lg`, `input-xl`
- Typography: `text-*`, `text-sm|base|lg|xl`, `text-mono`
- Icons: `icon`, `icon-<name>`, `icon-xs|sm|lg|xl`
