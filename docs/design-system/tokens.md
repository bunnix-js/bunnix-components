# Tokens

Tokens live in `src/styles/variables.css` and include colors, spacing, typography, and control sizing. Override them in your app to theme globally.

Example:

```css
:root {
  --accent-color: #2563eb;
  --border-color: #e5e7eb;
  --base-radius: 12px;
}
```
